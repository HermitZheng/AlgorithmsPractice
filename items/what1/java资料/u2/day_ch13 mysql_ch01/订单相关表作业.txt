作业一、实现订单的数据库设计:
商品表product
商品编号p_id  主键自动编号
名称 p_name  不能为空
单价 p_price  不能为空  
数量 p_count   不能为空 
是否是最新商品 p_new tinyint类型 1是最新 0不是   不能为空
是否是推商品 p_t tinyint类型 1是推荐   不能为空

用户表users
用户编号uid 主键 自动编号
用户名  uname   不能为空
密码 upwd   不能为空  默认是123456
真实姓名  name
电话 tel

订单表order 
订单编号  o_id 自动编号 主键
用户编号  u_id 外键 引用用户表的主键
送货地址  address 
收货人  name
收货人电话 tel

订单明细表items  [实现订单与商品的多对多关系]
明细编号id 主键自动编号 1开始
订单编号o_id 外键  引用订单表的主键
商品编号p_id 外键  引用商品表的主键
购买数量 count 默认为 1
